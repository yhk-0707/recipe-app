# recipe-app

このプロジェクトは、HTML/CSS/JavaScriptのレシピメモ用アプリを元に、Reactから発展してNext.js学習を進めるための構成です。

## ディレクトリ構成

- `index.html`, `detail.html`, `recipes.html`, `register.html`: 既存のHTML版ページ
- `style.css`: 共通スタイル
- `src/`: 既存のJavaScriptロジック
- `md/`: 学習メモ・学習計画
- `react-learning/`: React学習用の作業ディレクトリ

## 学習の進め方

1. まずは既存のHTML/CSS/JS版を理解する
2. その後、`react-learning/` でReactに置き換えていく
3. 1つずつ小さな変更をコミットする
