# レシピアプリについて

このプロジェクトは、レシピを検索・閲覧・登録できるシンプルなWebアプリです。

## できること

- レシピ一覧の表示
- レシピの詳細表示
- レシピ検索
- レシピ登録
- 登録済みレシピの管理

## 画面構成

- トップページ: レシピの概要や入口
- 一覧ページ: 登録済みレシピを一覧表示
- 詳細ページ: レシピの内容を表示
- 登録ページ: 新しいレシピを追加

## 使い方

1. ブラウザで index.html を開く
2. 画面上のリンクから各ページへ移動する
3. レシピを検索・閲覧・登録する

## 起動方法

ブラウザで直接開いても動作しますが、ローカルサーバーで開くとより安定します。

### 例: Python でサーバーを立てる

```bash
python -m http.server 8000
```

その後、ブラウザで http://localhost:8000/ を開きます。

## 開発の背景

このアプリは、HTML/CSS/JavaScriptで作られた元の実装をベースにしており、React学習のための置き換え対象として使っています。

## 主要ファイル

- index.html: トップページ
- detail.html: 詳細ページ
- recipes.html: 一覧ページ
- register.html: 登録ページ
- style.css: 共通スタイル
- src/: JavaScriptの処理ロジック
